## 视觉辅助

| 选项 | 说明 |
|------|------|
| 色盲模式 | 属性条使用图案纹理+颜色双重编码 |
| UI缩放 | 0.75x / 1.0x / 1.25x / 1.5x |
| 高对比度模式 | 增强 UI 元素对比度 |
| 字幕 | 所有音效配有文字描述选项 |
| 伤害数字大小 | 小 / 中 / 大 |

## 操控辅助

| 选项 | 说明 |
|------|------|
| 按键自定义 | PC端所有按键可重映射 |
| 闪避辅助 | 可开启自动闪避（降低伤害但不无敌） |
| 攻击辅助 | 可开启自动瞄准最近敌人 |
| 长按切换 | 格挡/奔跑从按住切换为点击切换 |

## 游戏性辅助

| 选项 | 说明 |
|------|------|
| 属性衰减调节 | 可调整饥饿/精神消耗速率 (50%-150%) |
| 死亡惩罚 | 可选择"软死亡"（丢失50%物品而非全部） |

**重要**：辅助选项的开启不影响遗产点数获取，但在图鉴中标记为"辅助模式"。
