## 情感曲线（单次游戏会话）

```
情感状态
  ▲
  │  😰恐惧          😱极度恐惧
  │    │                │
  │    ▼                ▼          😰恐惧
  │ ──────→ 🌙入夜 ──────→ 💀危险 ──────→ ☀️黎明 → 😌安心
  │                                            │
  │ 😤紧张                                     │
  │    ▼                                       ▼
  │ ──────→ ⚔️战斗 ──────→ 🏆胜利 → 😄成就感
  │                                            │
  │ 😢悲伤                                     │
  │    ▼                                       │
  │ ──────→ 💀死亡 ──────→ 📊回顾 → 🔄重来
```

## 核心情感目标

| 情感 | 触发条件 | 设计手段 |
|------|---------|---------|
| **孤独感** | 全程 | 无NPC对话、无背景音乐填充、广袤世界 |
| **恐惧感** | 夜间/黑暗 | 有限可视范围、未知声音、精神值系统 |
| **紧张感** | 战斗/低HP | 快节奏音乐、屏幕泛红、心跳音效 |
| **成就感** | Boss击败/发现 | 音乐高潮、屏幕特效、稀有物品 |
| **好奇心** | 探索 | 远处光点、可疑声音、未探索区域 |
| **悲伤感** | 死亡/叙事 | 音乐变化、环境叙事内容 |
| **满足感** | 建造/升级 | 视觉反馈、音效确认、数值成长 |

## 恐惧层次设计

```
第一层：不确定恐惧
  → "那里有什么？" — 黑暗中的声音、模糊的影子
  → 设计手段：有限视野、随机音效、精神值幻觉

第二层：生存恐惧  
  → "我要死了" — HP低下、资源耗尽、被追赶
  → 设计手段：UI警告、心跳音效、屏幕效果

第三层：存在恐惧
  → "这个世界怎么了？" — 叙事碎片揭示的真相
  → 设计手段：环境叙事、回声幻象

第四层：失去恐惧
  → "一切都白费了" — 永久死亡机制
  → 设计手段：死亡统计、遗产系统
```
