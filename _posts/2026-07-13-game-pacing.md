## 节奏触发器

| 阶段 | 紧张度 | 触发条件 | 游戏事件 |
|------|--------|---------|---------|
| 起步 | 中 | 游戏开始 | 基础教学，资源充足 |
| 第一个夜晚 | 高 | 第一天入夜 | 首次黑暗恐惧 |
| 黎明后 | 低 | 第二天黎明 | 放松探索 |
| 首次遭遇 | 高 | 遇到第一个敌人 | 学习战斗 |
| 稳定期 | 中低 | Day 3-5 | 建立据点 |
| Boss 战 | 极高 | 发现Boss | 紧张战斗 |
| 战后 | 低 | Boss击败 | 丰厚奖励 |
| 季节变化 | 高 | 进入冬季 | 环境压力骤增 |

## 安全区设计

| 安全区类型 | 条件 | 效果 |
|-----------|------|------|
| 篝火旁 | 篝火点燃中 | 精神回复，怪物回避(5格) |
| 帐篷内 | 帐篷建造完成 | 完全安全，可睡眠 |
| 石墙堡垒 | 石墙围合 | 高级安全，敌人无法进入 |

## 阶段性目标系统（隐性指引）

游戏不提供显式任务列表，通过「直觉提示」暗示下一步：

```typescript
interface IntuitionHint {
  conditions: (state: PlayerState) => boolean;
  hint_type: 'visual' | 'audio' | 'ui';
  content: string;
}

// 示例
const INTUITION_HINTS = [
  { conditions: (s) => s.hunger < 30, hint_type: 'visual',
    content: '附近的浆果丛微微发光' },
  { conditions: (s) => s.sanity < 25, hint_type: 'visual',
    content: '远处的篝火光芒更加醒目' },
];
```
