## 叙事碎片系统

游戏不使用对话和任务系统，通过环境叙事传递信息：

```typescript
interface NarrativeFragment {
  id: string;
  type: 'inscription' | 'journal' | 'mural' | 'remains' | 'echo';
  position: { x: number; y: number } | 'random_in_biome';
  biome: string;
  discover_radius: number;
  title: string;
  content: string;
  lore_chain?: string;
  required_fragments?: string[];
}
```

## 叙事碎片类型

| 类型 | 表现形式 | 获取方式 |
|------|---------|---------|
| 石碑铭文 | 古老石碑文字 | 探索遗迹 |
| 旅人日记 | 前人日记页 | 搜索营地/尸体 |
| 壁画 | 洞穴/遗迹壁画 | 探索地下区域 |
| 遗骸 | 带有物品的骸骨 | 探索隐秘区域 |
| 回声 | 精神值低时的幻象 | 低精神值+特定位置 |
