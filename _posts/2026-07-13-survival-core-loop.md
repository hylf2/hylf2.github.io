## 基础循环

```
探索世界 → 发现资源 → 采集/拾取
    ↑                        ↓
继续探索 ← 消耗物品 ← 合成/建造/食用
```

## 资源循环设计

**输入（获取）**：
- 采集（树木、石头、草药）
- 狩猎（肉类、皮毛、骨头）
- 探索（遗迹物品、宝箱）
- 战利品（击杀掉落）

**处理（转化）**：
- 合成台：原材料 → 工具/武器
- 烹饪：生食 → 熟食
- 建造：材料 → 建筑/设施

**输出（消耗）**：
- 食物：恢复 HP/Hunger
- 工具耐久：随使用消耗
- 建筑耐久：随时间/攻击消耗
- 光源燃料：篝火/火把持续消耗

## 单次游戏会话预期循环

```
第一阶段（0-5 分钟）：紧急生存
  → 采集基础资源（树枝、石头、浆果）
  → 制作石斧、火把
  → 寻找食物和水源

第二阶段（5-15 分钟）：建立据点
  → 选择营地位置
  → 建造篝火、简易庇护所
  → 制作基础工具和武器
  → 储备食物

第三阶段（15-30 分钟）：探索扩张
  → 探索周围区域
  → 收集稀有资源
  → 遭遇并战斗
  → 发现新区域/地标

第四阶段（30+ 分钟）：挑战与进阶
  → 准备Boss战
  → 制作高级装备
  → 挑战Boss
  → 获取奖励推进
```
