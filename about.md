## 项目简介

**荒野永夜**（Wild Eternal Night）是一款硬核野外生存 · 开放世界沙盒 · 动作冒险 · Roguelike 游戏。

- **目标平台**：PC (Windows) / 微信小程序
- **视角**：2.5D 等轴测（Isometric）
- **美术风格**：手绘卡通风 + 暗黑奇幻

## 设计理念

### 设计支柱

1. **残酷的生存感** — 世界不会怜悯你。每一滴水、每一口食物都至关重要。
2. **未知的恐惧** — 没有地图标记，没有任务指引，黑暗永远意味着危险。
3. **有意义的死亡** — 每次死亡都是一个故事的终结，但也开启了新的可能。
4. **发现的喜悦** — 在绝望中发现一处遗迹、一件稀有物品时的纯粹快乐。
5. **成长的满足** — 不是角色数值成长，而是玩家自身知识与技巧的成长。

### 核心体验循环

```
紧张 → 探索 → 发现 → 挑战 → 成就 → 失去 → 重生
```

## 文档体系

本站文档基于 **MDA（Mechanics-Dynamics-Aesthetics）框架** 扩展，涵盖六大设计支柱：

- **核心机制与规则** — 玩家做什么，规则是什么
- **系统与交互动态** — 运行中的系统、平衡、循环
- **叙事与世界观** — 故事、世界构建、设定
- **美学与感官体验** — 视觉、音频、情感设计
- **界面与用户体验** — UI/UX、控制、反馈
- **技术与平台** — 引擎、性能、约束

## MDA 映射

| 层次 | 设计要素 |
|------|---------|
| **Mechanics** | 三属性管理、资源采集、合成制作、动作战斗、永久死亡、程序化世界生成 |
| **Dynamics** | 资源稀缺驱动探索、危险驱动谨慎、知识驱动进步、Boss驱动目标感 |
| **Aesthetics** | 生存紧张感、探索好奇心、成就满足感、恐惧与敬畏、孤独感 |
